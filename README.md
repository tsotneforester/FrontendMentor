### Prerequisites

npm (for React Projects)

```sh
npm install npm@latest -g
```

### Installation & Usage

For React Projects
  1. In REACT folder install NPM package
  ```sh
  npm install
  ```
  2. Run React App
  ```sh
  npm start
  ```

For Vite Projects
  1. In VITE folder install NPM package
  ```sh
  npm install
  ```
  2. Run Vite App
  ```sh
  npm run dev
  ```
